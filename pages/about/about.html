Pengalaman Pengelolaan Program

  • Program penjangkauan MSM dan TG dukungan Global fund 2010 - 2015
  • Program penjangkauan WPS dukungan Global Fund 2010 - 2013
  • Program penjangkauan Pengguna Narkoba Suntik dukungan BNN tahun 2013
  • Program penjangkauan MSM dan TG dukungan Global fund 2016 - 2020
  • Program penjangkauan MSM dan TG dukungan Epic fund 2021 - 2022
  • Program penjangkauan MSM dan TG dukungan Global fund 2022 - 2023
  • Program penjangkauan WPS dukungan Global fund 2022 - 2023

  • Program penjangkauan MSM dan TG dukungan Global fund

    Program penjangkauan HIV untuk komunitas MSM dan TG yang didanai oleh global fund, dengan ara intervensi Kab. Tangerang, Kota Tangerang dan Tangerang Selatan. periode program 2010 - 2015.

  • Program penjangkauan WPS dukungan Global Fund 2010 - 2013

    Program penjangkauan HIV untuk komunitas WPS yang didanai oleh global fund, dengan ara intervensi Kab. Tangerang dan Kota Tangerang. periode program 2010 - 2013.

  • Program penjangkauan Pengguna Narkoba Suntik dukungan BNN tahun 2013

    Program penjangkauan HIV untuk komunitas Pengguna narkoba jarum suntik yang didanai oleh Badan Narkotika Nasional (BNN), dengan ara intervensi Kab. Tangerang dan Kota Tangerang. periode program 2013.

  • Program penjangkauan MSM dan TG dukungan Global fund 2016 - 2020

    Program penjangkauan HIV untuk komunitas MSM dan TG yang didanai oleh global fund, dengan ara intervensi Kab. Tangerang dan Kota Tangerang, Kab.Serang, Kota Serang, Kota Cilegon, Kab.Lebak, dan Kab.Pandeglang. periode program 2016 - 2020.

  • Program penjangkauan MSM dan TG dukungan Global fund 2016 - 2020

    Program penjangkauan HIV untuk komunitas MSM dan TG yang didanai oleh global fund, dengan ara intervensi Kab. Tangerang dan Kota Tangerang, Kab.Serang, Kota Serang, Kota Cilegon, Kab.Lebak, dan Kab.Pandeglang. periode program 2016 - 2020.

  • Program penjangkauan dan pendampingan MSM dan TG dukungan USAID 2021 - 2022

    Program penjangkauan dan pendampingan HIV untuk komunitas MSM dan TG yang didanai oleh global fund, dengan ara intervensi Kab. Tangerang dan Kota Tangerang. periode program 2021 - 2022.

  • Program penjangkauan MSM dan TG dukungan Global fund 2022 - 2023

    Program penjangkauan HIV untuk komunitas MSM dan TG yang didanai oleh global fund, dengan ara intervensi Kab. Tangerang, Kota Tangerang, Kota Serang, Kab.Serang dan Kota Cilegon. periode program 2022 - 2024.

  • Program penjangkauan WPS dukungan Global fund 2022 - 2023

    Program penjangkauan HIV untuk komunitas WPS yang didanai oleh global fund, dengan ara intervensi Kab. Tangerang, Kota Tangerang dan Tangerang Selatan. periode program 2022 - 2024.
